hexo播放器

2017-08-08

1.APlayer.min.js

<script src="https://cdn.bootcss.com/aplayer/1.6.0/APlayer.min.js"></script>

2.player.js

<script src="https://api.i-meto.com/music/player.js"></script>

将APlayer.min.js代码插入到/layout/common/header.ejs里面
将player.js代码插入到/layout/common/footer.ejs里面

上述准备工作做完后，只需在插入音乐的地方添加：

<div class="aplayer" data-id="373559" data-server="xiami" data-type="song"></div>

其中的参数如下，加粗表示必须填写

  • data-id: 歌曲/专辑/歌单 ID

      • data-server: 音乐平台，支持如下参数

  • netease （网易云音乐）

  • tencent （qq音乐）

  • xiami （虾米音乐）

  • kugou （酷狗音乐）

  • baidu （百度音乐）

        • data-type: 请求类型，支持如下参数

    • song （单曲）

    • album （专辑）

    • playlist （歌单）

    • search （搜索）

          • data-mode: 播放模式

      • random （随机）

      • single （单曲）

      • circulation （列表循环）

      • order （列表）

            • data-autoplay: 自动播放

        • false

        • true


            • 更多参数详见 https://aplayer.js.org/docs/#/?id=options
